# ISTE240 Docker Project – Roasted Duck

This project containerizes my previous ISTE240 PHP website using Docker and Docker Compose.

## How to Run

1. Make sure Docker Desktop is installed
2. Navigate to the project folder
3. Run:

   docker compose build
   docker compose up

4. Open a browser and go to:

   http://localhost:8080

## Technologies Used

- PHP 8.2
- Apache
- Multi-stage Docker build
- Docker Compose
